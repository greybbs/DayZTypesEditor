# DayZ Types Editor

Веб-приложение для настройки и валидации файла **types.xml** в DayZ (спавн лута).

## Возможности

- **Загрузка** — открыть `types.xml` с диска или вставить XML в поле ввода
- **Список типов** — поиск по имени и фильтр по категории
- **Редактирование** — изменение nominal, lifetime, restock, min, quantmin, quantmax, cost, flags, category, usage, value
- **Валидация** — проверка числовых полей, флагов (0/1), обязательных полей и связи quantmin/quantmax
- **Экспорт** — скачать обновлённый `types.xml`

## Запуск

Откройте `index.html` в браузере (двойной клик или через локальный сервер).

Для больших файлов (как ваш пример) загрузка и разбор могут занять несколько секунд.

## Формат types.xml

- Корневой элемент: `<types>`
- Каждый тип: `<type name="...">` с полями nominal, lifetime, restock, min, quantmin, quantmax, cost, flags, category, опционально usage и value

Валидация проверяет: числа в допустимых диапазонах, флаги 0/1, quantmax ≥ quantmin при не -1.
